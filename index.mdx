Writer is a local-first desktop markdown editor designed for folders of plain-text notes, documentation, and personal wikis. Every document lives on your disk as a standard `.md` file — no proprietary format, no sync account required. Open any folder as a workspace, write in a focused editor, and navigate your files through a clean sidebar.

## What Writer does

Writer keeps your writing workflow entirely on your machine. You open a folder — an Obsidian vault, a docs repo, a personal notes directory — and Writer shows you the markdown files inside it. The editor renders headings, lists, tables, task checkboxes, code blocks, and Mermaid diagrams as you type. Files are saved automatically, atomically, with no data loss on crash.

## Get started

<Steps>
  <Step title="Download Writer">
    Grab the latest signed macOS DMG from the [GitHub releases page](https://github.com/joelbqz/writer-computer/releases/latest).
  </Step>
  <Step title="Open a folder">
    Launch Writer and click **Add Folder** on the welcome screen, or drag a folder onto the Dock icon.
  </Step>
  <Step title="Start writing">
    Click any `.md` file in the sidebar to open it, or press `Cmd+N` to create a new note. Type `# ` to add a title and begin writing immediately.
  </Step>
  <Step title="Navigate with the command palette">
    Press `Cmd+P` to open the command palette and jump to any file or run any command without leaving the keyboard.
  </Step>
</Steps>

<Tip>
  Install the `writer` shell command from **Writer → Shell Command: Install 'writer' Command in PATH** to open folders and files directly from the terminal with `writer .` or `writer notes.md`.
</Tip>
